# VK Messenger Custom Themes
Темы для VK Messenger. Ночные, цветные.

Переместить все файлы по пути

* MacOS: `~/Library/Application Support/VK/themes`
* Linux: `~/.config/VK/themes`
* Windows: `%AppData%/vk/themes`
