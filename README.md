# VK Messenger Custom Themes
Темы для VK Messenger. Ночные, цветные.

## MacOS
Файлы переместить по пути
```
~/Library/Application Support/VK/themes
```

## Windows
1. Windows + R
2. Ввести %AppData%
3. Положить файлы в папку VK/themes
